import random
brandlist = [‘华为’, ‘苹果’, ‘诺基亚’, ‘oppo’, ‘小米’]
random.seed(0)
name = brandlist[random.randint(0, 4)]
print(name)

random.seed()
改变随机生成器的种子，使得每次生成的随机数相同

random.random()
生成0~1之间的随机浮点数

random.uniform（a, b）
指定范围内的随机浮点数

random.randint(a, b)
[a, b]内的随机整数

random.randrange(start, stop, step)
有梯度的序列中随机取一个随机数

random.shuffle()
