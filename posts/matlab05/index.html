第五章 数据分析与多项式计算

1.统计分析
A = [1, 3, 5, 7, 9; 8, 6, 4, 2, 0];
max_value = max(A, [], 2) % 行的最大值, min同理
sum_value = sum(A, 2) % 行的求和
prod_value = prod(A, 2) % 行的求积
mean_value = mean(A, 2) % 行的均值
median_value = median(A, 2) % 行的中值
sort_value = sort(A, 2, ‘descend’) % 行的降序排序
2.多项式计算
% 多项式1; x³ -2x² + 5x +3
% 多项式2: 6x² -1
a = [1, -2, 5, 3];
b = [0, 6, 0, -1];
sum = a + b
prod = conv(a, b)
% deconv(a,b)
polyder(a)
polyder(a, b)
polyder(prod)
polyval(a, 2)
roots(a)
poly(ans)
4.曲线插值与曲线拟合
x = [1, 2, 3, 4, 5];
y = [3, 6, 9, 12, 15];
interp1(x, y, 9,’spline’)
spline(x, y, 9)
x = linspace(0, 2 * pi, 100);
y = sin(x);
polyfit(x, y, 3)
y1 = polyval(ans, x)
plot(x, y, x, y1)
