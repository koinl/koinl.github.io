第八章 数据库后台编程技术

存储过程

  1. 创建:create proc proc_name

    　 　@var type,@var1 type output

    　 　as

    　 　begin

    　 　sql_statement

    　 　end
  2. 执行:exec proc_name @var=

    函数

  3. 分类:标量函数:create function fun_name(@var type)

    　　 　　 　　returns type

    　　 　　 　　as

    　　 　　 　　begin

    　　 　　 　　sql_statement

    　　 　　 　　end

    　 　表值函数:内嵌表值函数:create function fun_name(@var type)

    　　　　　　　　 　　 　 　returns table

    　　　　　　　　 　　 　 　as

    　　　　　　　　 　　 　 　begin

    　　　　　　　　 　　 　 　return(sql_statement)

    　　　　　　　　 　　 　 　end

    　　 　　 　多语句表值函数:create function fun_name(@var type)

    　　　　　　　　 　　 　 　returns @tb_name table(col_name)

    　　　　　　　　 　　 　 　as

    　　　　　　　　 　　 　 　begin

    　　　　　　　　 　　 　 　sql_statement

    　　　　　　　　 　　 　 　return

    　　　　　　　　 　　 　 　end

    触发器

  4. 创建:create trigger tri_name

    on tb_name

    {for | instead of} {insert | update | delete}

    as

    begin

    sql_statement

    end

游标

  1. 声明:declare cur_name cursor

    　 　for sql_statement
  2. 打开:open cur_name
  3. 提取:fetch first | last | next | prior | absolute | relative
    from cur_name
  4. 关闭:close cur_name
  5. 释放:deallocate cur_name
  6. 全局变量:@@fetch_status
