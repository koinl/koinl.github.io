第某章 逻辑卷管理

2022-05-10 Linux

逻辑卷的管理, 嘛, 磁盘空间的动态管理.
可分为创建与管理逻辑卷.

一, 创建逻辑卷

首先, 需要安装 LVM2, yum install lvm2
第一步: 建立主分区, 命令为 fdisk /dev/sda n p 4... 第二步: 修改文件类型: 命令为 接上述 Command (m for help): t 4 8e第二步: 初始化分区 (创建物理卷 PV ), 命令为pvcreate /dev/sda2第三步: 创建卷组 VG, 命令为vgcreate -s 8M dd /dev/sda2第四步: 创建逻辑卷 LV, 命令为lvcreate -n yy -L 100M koinl_vg第五步: 格式化, 命令为mkfs -t ext4 /dev/dd/yy第六步: 挂载, 命令为mount /dev/dd/yy /tmp/mnt`

二, 管理逻辑卷

1. 查看 LVM 状态

pvs, vgs, lvs
pvdisplay, vgdisplay, lvdisplay

2. 扩大, 缩小逻辑卷及文件系统

第一步: 查看卷组的剩余空间以保证可以进行扩大: vgdisplay 以查看 Free PE .
第二步: 进行扩展: lvextend -L +10M /dev/dd/yy ; 扩展逻辑边界: resize2fs /dev/dd/zz
第三步: 进行缩减: lvreduce -L 10M /dev/dd/yy.

3. 删除逻辑卷

前提是先要卸载 lv 文件系统
umount /tmp/mnt
e2fsck -f /dev/dd/yy
lvreduce -L 10M /dev/dd/yy

pv vg lv
扫描 scan
创建 create
查看状态 display
删除 remove
缩进 reduce
扩展 extend

例: 创建一个VG (名为 wgroup) 和LV ( lv名为 wshare), PE大小8M, LV中PE个数100,文件系统格式为ext4
解: 先创建一个主分区 sda4, 再然后创建 PV: pvcreate /dev/sda4 , 第三步创建 VG: vgcreate -s 8M wgroup /dev/sda4 , 第四步创建 LV: lvcreate -L 100 -n wshare /dev/wgroup , 第五步格式化: mkfs -t ext4 /dev/wgroup/wshare .

例2: 新建一个LV, 名称为home, 大小为1000M, 格式为ext4, 将home逻辑卷从1000M扩容到1500M
解: 假定 home 改名为上题 wshare.
扩容: lvextend -L +500M /dev/wgroup/wshare 或者 lvextend -L 1500M /dev/wgroup/wshare .
