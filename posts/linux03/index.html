第三章 存储和逻辑卷管理

分类： Linux
标签：

日期：2022-06-09 11:18:06

新建完分区后，需重新读取内核。另，若挂载则需先进行格式化
对于一个硬盘，最多可以设置四个主分区。
三个主分区，一个扩展分区
fdisk /dev/sda p查看临时盘信息 n创建 d删除 t修改id w保存
Linux 83
Linux lvm 8e
Linud swap 82

一. 存储管理

1. 磁盘分区

1) 查看磁盘分区

fdisk -l
lsblk -f(设备挂载情况）

2) 添加分区

fdisk /dev/sda，n，p，4，↲，+512M，w，partprobe

3) 删除分区

fdisk /dev/sda，d，2，w，partprobe

2. 磁盘挂载与卸载

1) 磁盘挂载

注：所挂载目录需存在
mkfs -t ext4 /dev/sda4
mount /dev/sda4 /sta/knl

2) 磁盘卸载

umount /dev/sda4

二. 交换分区

fdisk /dev/sda，n…，t 82 w，partprobe
mkswap /dev/sda6
swapon /dev/sda6
vi /etc/fstab
/dev/hda6 swap swap defaults 0 0
mount -a

or

dd if=/dev/zero of=/opt/swapfile bs=1M count=1000 (bs blocksize ，每个块大小为1M.count=2048。则总大小为2G的文件。创建一个1G的文件作为交换分区使用)
mkswap /opt/swapfile
swapon /opt/swapfile
vi /etc/fstab
/opt/swapfile swap swap defaults 0 0
mount -a
